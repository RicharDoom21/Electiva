Security System Study 

Security System Study es una plataforma web que ofrece servicios de software de seguridad para instituciones educativas, con el objetivo de garantizar la seguridad y controlar el ingreso y salida de estudiantes, docentes, celadores y empleados de servicio.

learn more

Los

Objetivos

Estabilidad

Ofrecer un software escalable y adaptable que brinde seguridad y control en el área estudiantil.

Versatilidad

El objetivo es que la plataforma sea práctica y versátil, para poder ser utilizada en diferentes dispositivos de seguridad..

Seguridad

Brindar un servicio de seguridad eficiente, compatible con diferentes sistemas de seguridad y que garantice la integridad de la comunidad estudiantil.

learn more

Publico dirigido

Dirigido a instituciones educativas que deseen mejorar su seguridad y tener un registro de quienes ingresan y salen de su plantel.

También ofrecemos asesoramiento y soporte técnico para garantizar el correcto funcionamiento del software.

Soporte técnico 

Nuestro mensaje es emocional, ya que queremos transmitir seguridad y confianza a los estudiantes y docentes. El tono de nuestra comunicación es amigable y cercano, para establecer una relación de confianza con nuestros clientes.

Son Ricardo Reina y Bayron Serrato, expertos en seguridad informática y desarrollo de software. Contamos con un presupuesto de 200,000 pesos para el desarrollo y lanzamiento de Security System Study.
